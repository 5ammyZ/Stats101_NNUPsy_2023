某心理学家从受过专门训练的儿童中随机抽取64名儿童进行韦克斯勒儿童智力测验(\(\mu_0\) = 100， \(\sigma_0\) = 15)。结果发现，这64名儿童的平均智商为105。 问：能否认为这些接受了该项训练的儿童的智力高于其所在年龄组儿童的一般水平？

    1. 提出假设

  • 一个总体是正常孩子，

  • 另一个总体是超过正常智力水平的孩子

    1. 根据虚无假设\(H_0\)，选择相应的统计模型。

    • 我们假定样本量为64的正常”儿童群体的平均智力”呈现正态分布，可使用z检验;

    • 在使用z检验之前，需要检查数据是否满足z检验的预设。

  • z检验的预设为：

    • 总体为正态分布;

    • 样本量大于30;

    • 样本平均数的抽样分布为正态分布

    • 指定\(\alpha=0.05\)(拒绝零假设犯错的可能性)；

    • 单侧检验的临界点\(Z_{0.05}=1.645\)

    • 注意：显著性水平意味着犯错误的可能性

  • \(\sigma_{\bar{X}} = \frac{\sigma_0}{\sqrt{n}}=\frac{15}{\sqrt{64}}=1.875,Z=\frac{\bar{X}-\mu_0}{\sigma_{\bar{X}}}=\frac{105-100}{1.875}=2.67\)

    • 受训的儿童与正常孩子的差异并非特别大。

单样本t检验



一般来说，人的视觉反应时符合正态分布,且某心理学家研究发现，普通飞行员的平均视觉反应时为170ms，某人随机抽取25名飞行员进行测定，结果发现其平均视觉反应时为175ms，标准差\(S_{n-1}\) 为15。问：能否根据测试结果而否定该心理学家的结论？



问题1：该样本飞行员的视力和普通飞行员是否存在差异？

问题2：该样本飞行员的视力相比普通飞行员的视力差多少？



问题1：该样本飞行员的视力和普通飞行员是否存在差异？（假设检验）

问题2：该样本飞行员的视力相比普通飞行员的视力差多少？（参数估计）



我们该怎么计算\(𝐻_0\)成立下，该样本飞行员的视力超过正常总体均值的概率\(p\)



  • t 检验的预设为

    • 总体为正态分布

    • 样本平均数的抽样分布为t 分布

基本步骤



  • 1.提出假设

  • 该样本飞行员的视力和普通飞行员是否存在差异，提出相应的虚无假设\(𝐻_0\)和研究假设\(𝐻_1\)，选择使用单侧检验。

    • \(𝐻_0\)：该样本飞行员的视力和普通飞行员没有差异， $_0 = _1 $

    • \(𝐻_1\)：该样本飞行员的视力优于普通飞行员， $_0 < _1 $

假定存在两个总体

一个总体是普通飞行员

另一个总体是样本飞行员



    1. 根据虚无假设𝐻_0，选择相应的统计模型。

    • 我们假定样本量为25的样本飞行员”的视觉反应时”呈现正态分布，可使用t检验。

    • 在使用t 检验之前，需要检查数据是否满足t检验的预设。



    1. 确定显著性水平\(\alpha\)，\(\alpha\)确定后，否定域也随之被确定了。

    • 指定\(\alpha\) = 0.05（拒绝零假设犯错的可能性）。

    • 单侧检验的临界点 \(df\) = 24, \(t_{0.05}=1.711\)



  • 4.基于\(H_0\)所设定的统计模型，计算检验统计量的值。

  • \(S_{\bar{X}} = \frac{S_0}{\sqrt{n}}=\frac{15}{\sqrt{25}}=3,t=\frac{\bar{X}-\mu_0}{\frac{S_{n-1}}{\sqrt{n}}}=1.67\)



  • 5.做出决策

    • 单侧检验的临界点\(t_{0.05}=1.711\);

    • \(t = 1.67 < t_{0.05}\)

    • 效应量\(d = \frac{\bar{X}-\mu_0}{S_{n-1}}=\frac{175-170}{15}=0.3\)

    • 在\(\alpha=0.05\)显著性水平上不能拒绝虚无假设，即\(\mu_{\bar{X}}<\mu_0\)



样本飞行员的平均视觉反应时为175ms ，普通飞行员的平均视觉反应时为170ms，我们可以认为两者的差异没有达到统计上的显著水平。



抽取的飞行员的平均视觉反应时与普通飞行员的平均视觉反应时相 比较差异没有统计学意义\((t (24) = 1.67, p > 0.05\), Cohen’s d = 0.3, 95% CI : [-0.07 , 0.74]), 可以认为抽取的飞行员与普通飞行员来自同一总体。

独立样本的差异检验



某心理学家从南方地区的7岁儿童中随机抽取36名男童和34名女童，平均身高分别为：男童125cm,女童127cm，以往资料显示，该地区7岁男童身高的标准差为5，女童身高的标准差为6，能够根据这次抽样测量的结果得出”该地区7岁男女儿童身高有显著差异”的结论？



问题1：该地区7岁男女儿童身高是否存在差异？

问题2：该地区7岁男女儿童身高差多少？



问题1：该地区7岁男女儿童身高是否存在差异？（假设检验）

问题2：该地区7岁男女儿童身高差多少？（参数估计）



我们该怎么计算\(H_0\)成立下，该地区7岁女童身高与男童身高表现出差异的概率\(p\)



  • 1.提出假设。

  • 根据问题”该地区7岁男女儿童身高是否存在差异” ，提出相应的虚无假设\(H_0\)和研究假设\(H_1\)，选择使用双侧检验。

    • \(𝐻_0\)：该地区7岁男女儿童身高没有差异， \(\mu_1 = \mu_2\) 。

    • \(𝐻_1\)：该地区7岁男女儿童身高存在差异， \(\mu_1 \ne \mu_2\) 。

联合方差

对具有不同均值的各种人群的固定共同方差的估计

  • 给定一系列样本方差

\[s_i^2 = \frac{1}{n_i-1}\sum_{j=1}^{n_i}(y_j-\bar{y_i})^2\]

  • 假定样本大小一致

\[s_p^2 = \frac{\sum_{i=1}^ms_i^2}{m}\]

  • 如果样本量不同，加权平均

\[s_p^2 = \frac{\sum_{i=1}^m (n_i-1)s_i^2}{\sum_{i=1}^m (n_i-1)}\]



  • 联合方差能够更好地估计标准差和标准误差

  • 联合方差假定方差齐性



    1. 根据虚无假设\(H_0\)，选择相应的统计模型。

    • 我们假定该地区7岁男女儿童身高呈现正态分布，可使用z检验。

    • 在使用z 检验之前，需要检查数据是否满足z检验的预设。



    1. 确定显著性水平\(\alpha\)，\(\alpha\)确定后，否定域也随之被确定了。

    • 指定\(\alpha = 0.05\)（拒绝零假设犯错的可能性）。

    • 双侧检验的临界点 \(𝑍_{0.05/2}=1.96\)



    1. 基于\(H_0\)所设定的统计模型，计算检验统计量的值。
    • \(Z=\frac{\bar{X_1}-\bar{X_2}}{\sqrt{\frac{\sigma^2_1}{n_1}+\frac{\sigma^2_2}{n_2}}}=\frac{125-127}{\sqrt{\frac{5^2_1}{36}+\frac{6^2_2}{34}}}=1.51\)

  • 5.做出决策。

    • 双侧检验的临界点\(Z_{0.05/2}=1.96\)

    • \(𝑍 = 1.51 <Z_{0.05/2}\)

    • \(Cohen’ d = \frac{\bar{X_1}−\bar{X_2}}{\sqrt{S_p}}= 0.36\)

    • 在\(\alpha = 0.05\)显著性水平上不能拒绝虚无假设，即\(\mu_1 = \mu_2\)。

我们可以认为该地区7岁男童与女童身高两者的差异没有达到统计上的显著水平。

结果呈现



抽取的该地区7岁男童与女童身高相比，差异没有统计学意义\((Z = 1.51, p > 0.05\), Cohen’s d = 0.36, 95% CI [-0.11, 0.83])。根据当前数据，我们无法推断该地区7岁男童与女童在总体身高上有差异。



从参加某区数学统考的高一学生中随机抽取男生60人，其平均成绩为78分，标准差为6；女生56人，其平均成绩为75分，标准差为5。假设男女生两总体的方差一致，问男女生的数学成绩有无显著差异？



  • 1.提出假设。

  • 根据问题”男女生的数学成绩是否存在差异” ，提出相应的虚无假设\(H_0\)和研究假设\(H_1\)，选择使用双侧检验。

    • \(𝐻_0\)：男女生的数学成绩没有差异， \(\mu_1 = \mu_2\) 。

    • \(𝐻_1\)：男女生的数学成绩存在差异， \(\mu_1 \ne \mu_2\) 。



    1. 根据虚无假设\(H_0\)，选择相应的统计模型。

    • 我们假定男女生的数学成绩呈现正态分布，可使用t检验。

    • 在使用t 检验之前，需要检查数据是否满足t检验的预设。



  • t 检验的预设为：

    • 观察间彼此独立

    • 两个总体均为正态分布

    • 两个总体具有相等的方差，即方差同质性

  • 总体正态，方差未知

  • 方差齐性检验：

    • \(n_1=60, n_2=56,S_1^2 = 6^2, s_2^2=5^2\)

    • \(F= \frac{S_{max}^2}{S_{min}^2 } = 36/25 =1.44\)

  • 分子自由度\(df_1=60−1=59\),分母自由度\(df_2=56−1=55\),

  • 查表双侧检验，\(F_{0.05/2}= 1.67\)，

  • \(F = 1.44 < 1.67, p > 0.05\)

  • 两个方差的差异未达0.05显著性水平，接受方差齐性假设



    1. 确定显著性水平\(\alpha\)，\(\alpha\)确定后，否定域也随之被确定了。

    • 指定\(\alpha = 0.05\)（拒绝零假设犯错的可能性）。

    • \(df=n_1+ n_2−2=114\),双侧检验的临界点 \(𝑡_{0.05/2}=1.984\)



    1. 基于\(H_0\)所设定的统计模型，计算检验统计量的值。
    • \(t=\frac{\bar{X_1}-\bar{X_2}}{\sqrt{\frac{n_1S^2_1+n_2S_2^2}{n_1+n_2-2}(\frac{n_1+n_2}{n_1n_2})}}=\frac{78-75}{\sqrt{\frac{60*6^2_1+56*5^2}{60+56-2}(\frac{60+56}{60*56})}}=2.89\)

  • 5.做出决策。

    • 双侧检验的临界点\(t_{0.05/2}=1.984\)

    • \(t = 2.89>t_{0.05/2}\)

    • 在\(\alpha = 0.05\)显著性水平上可以拒绝虚无假设，即\(\mu_1 = \mu_2\)。

我们可以认为男女生的数学成绩在0.05显著性水平上的差异显著。

  • \(t'=\frac{\bar{X_1}-\bar{X_2}}{\sqrt{\frac{S^1_2}{n_1-1}+\frac{S^2_2}{n_2-1}}}\)

配对样本的差异检验

某心理学家随机抽取了一小学36名刚入学的儿童进行韦氏智力测验(\(\sigma = 15\))，结果平均智商为110。一年后又对同组被试进行了重测，结果平均智商为115。已知两次智力测验结果的相关系数为r = 0.70。问：能否认为经过一年的教育及年龄增长，儿童的智商有了显著提高。



  • 1.提出假设。

  • 根据问题”经过一年的教育及年龄增长，儿童的智商是否存在提高” ，提出相应的虚无假设\(H_0\)和研究假设\(H_1\)，选择使用双侧检验。

    • \(𝐻_0\)：经过一年的教育及年龄增长，儿童的智商没有明显变化， \(\mu_1 = \mu_2\) 。

    • \(𝐻_1\)：经过一年的教育及年龄增长，儿童的智商显著提高， \(\mu_1 < \mu_2\) 。



    1. 根据虚无假设\(H_0\)，选择相应的统计模型。

    • 我们假定儿童韦氏智力测验分数呈现正态分布，可使用z检验。

    • 在使用z 检验之前，需要检查数据是否满足z检验的预设。



    1. 确定显著性水平\(\alpha\)，\(\alpha\)确定后，否定域也随之被确定了。

    • 指定\(\alpha = 0.01\)（拒绝零假设犯错的可能性）。

    • 单侧检验”的临界点”\(Z_{0.01}=2.33\)



    1. 基于\(H_0\)所设定的统计模型，计算检验统计量的值。
    • \(\sigma_{D_{\bar{X}}}=\sqrt{\frac{\sigma_1^2}{n}-2r\frac{\sigma_1}{n}\frac{\sigma_1}{n}+\frac{\sigma_2^2}{n}}\)

    \(=\sqrt{\frac{\sigma_1^2}{\sqrt{n}^2}-2r\frac{\sigma_1}{n}\frac{\sigma_1}{n}+\frac{\sigma_2^2}{\sqrt{n}^2}}\)

    \(=\frac{\sigma}{\sqrt{n}}\sqrt{2-2r}=1.936\)

    • \(Z = \frac{\bar{X_1}-\bar{X_2}}{\sigma_{D_{\bar{X}}}}\)

  • 5.做出决策。

    • 双侧检验的临界点\(t_{0.05/2}=1.984\)

零效应检验

零效应检验

我们能接受虚无假设(该样本飞行员的视力和普通飞行员没有差异)吗？



  • 在假设检验的框架下，p值表示在假定虚无假设为真的条件下出现当前数据的概率，p值并不能提供证据来接受虚无假设。

  • 若该心理学家想证明这些飞行员之间视力不存在差异，他该怎么办？(通过什么方法可以接受虚无假设)

在假设检验的框架下，假设表述如下：

\(𝐻_0\)：该样本飞行员的视力和普通飞行员没有差异， \(d = 0\)。

\(𝐻_1\)：该样本飞行员的视力和普通飞行员存在差异， \(d ≠ 0\)。

实际上，研究者关心的效应量是否非常小，以至可以忽略。因此，可将虚无假设的效应量拓展为一个区间, 这种方法叫做最小效应量检验：

\(𝐻_0\)：该样本飞行员的视力和普通飞行员没有差异， \(d = [-0.32, 0.32]\)。

\(𝐻_1\)：该样本飞行员的视力和普通飞行员存在差异， \(d > 0.32 或 d < -0.32\)。

Lakens, (2022). Improving Your Statistical Inferences. Chapter 9, https://doi.org/10.5281/zenodo.6409077. Built on 2022-05-29
许岳培, 陆春雷, …, 胡传鹏. (2022). 应用心理学



  • 我们可以设定一个足够小的，几乎可以认为效应为零的区间，用于评估零效应。

  • 这一区间即为”最小感兴趣的效应量区间”，简称为” 最小感兴趣区”（Smallest effect size of interest，SESOI）。

  • 目标效应量在SESOI内时，研究者可以认为效应量几乎为零，可以忽略不计。



等价检验则正好将最小效应量检验的\(𝐻_0\) 与\(𝐻_1\)所对应的效应区间对调， \(𝐻_1\)在区间之内，而\(𝐻_0\)在区间之外。

在等价检验的框架下，虚无假设表述如下：

\(𝐻_0\)：该样本飞行员的视力和普通飞行员存在差异， \(d > 0.32 或 d < -0.32\)。

\(𝐻_1\)：该样本飞行员的视力和普通飞行员没有差异， \(d = [-0.32, 0.32]\)。
